# 🎫 OBED Ticket 예매 시스템

- **모바일 중심**의 티켓 신청 시스템.  
- React + TypeScript + Vite 기반 프론트엔드, Node.js + Express + TypeScript + MySQL 기반 백엔드.  
- Firebase 인증 및 관리자 기능, Render/Vercel을 통한 무료 배포 구성.

## 🧪 기능 요약

### 사용자용
- 티켓 신청 (이름, 이메일, 학생/성인)
- 신청 결과 확인 (티켓 ID 기반)
- 모바일에 최적화된 UI

### 관리자용 (예정)
- 티켓 상태 확인 및 확정 처리
- Firebase 인증 기반 보호
